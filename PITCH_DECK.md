# DESIGN NOTES

## Color Scheme
- Primary: Electric Blue (#0066FF) - Arc blockchain
- Secondary: Neon Purple (#9D4EDD) - AI/Tech
- Accent: Gold (#FFD700) - USDC/Money
- Dark: #0A0A0A - Background

## Fonts
- Headlines: Bold, modern sans-serif (Montserrat, Inter)
- Body: Clean, readable (Open Sans, Roboto)
- Code: Monospace (Fira Code, JetBrains Mono)

## Visual Style
- Dark theme with bright accents
- High contrast for readability
- Use icons liberally
- Include actual app screenshots
- Add motion/animation in presentation

## Tips for Slides
1. Keep text minimal - use bullet points
2. One main idea per slide
3. Use large, readable fonts (24pt+ body, 36pt+ headlines)
4. Include visual elements on every slide
5. Practice presenting - aim for 3-5 minutes total
